Features of Roomba
We provide you the best features in Roomba!
Dashboard

Roomba comes with a fully functionnal dashboard that will allow you to custom the bot to perfectly fit your server. You'll can for example, change the prefix or even set a word banlist.

Music

With Roomba you can listen to music simply using an URL or using the automatic search tool. We're actually working on implementing Spotify and Soundcloud support.

Moderation

Roomba comes with a fully functionnal pack of moderation tools. All the commands that you could need like ban, mute, kick, clear, prune, warn and report are implemented.

Utilities

Roomba is a multi purpose bot, so you'll can find fun commands like memes command, informations commands, weather command, encrypting commands and even NSFW.
